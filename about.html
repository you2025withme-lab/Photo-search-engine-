Photo Search Engine

About This Project

Photo Search Engine is a lightweight and fast web application that allows users to upload an image and instantly search for visually similar images using Google, Bing, or Yandex.

Built entirely with HTML, CSS, and JavaScript, it works completely in the browser without storing any images on a server. The interface is simple, icon-based, and fully responsive for all devices.

This project is open-source and can be freely edited, customized, or shared.
